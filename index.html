⚙️ Configuration de la synchronisation
Pour synchroniser les données entre vos deux ordinateurs, vous avez besoin d'une clé API JSONBin (gratuit, 1 minute).
1. Allez sur jsonbin.io et créez un compte gratuit.
2. Dans votre dashboard JSONBin, allez dans API Keys → copiez votre Master Key (commence par $2b$10$…).
3. Collez-la ci-dessous et cliquez Connecter. La même clé sur les deux ordinateurs = données partagées.
Si vous avez déjà configuré un autre ordinateur, utilisez exactement la même clé pour partager les mêmes données.
